# Reflection — Async Calculator App

## 📌 Project Overview
This project is an Object-Oriented Async Calculator built using Dart.  
It demonstrates OOP principles, exception handling, and async/await usage.

---

## 🧠 What I Learned

### 1️⃣ Object-Oriented Programming (OOP)
- Created a Calculator class
- Used constructor to initialize values
- Implemented methods (add, subtract, multiply, divide)

This helped me understand encapsulation and clean code structure.

---

### 2️⃣ Async / Await
- Used Future<double>
- Applied async and await
- Simulated delay using Future.delayed()

I learned how Dart handles asynchronous operations.

---

### 3️⃣ Exception Handling
- Used try and catch blocks
- Threw custom exceptions (e.g., divide by zero)
- Prevented program crash from invalid input

This improved my understanding of safe programming.

---

## ⚠ Challenges Faced
- Handling nullable values
- Managing async return types
- Preventing division by zero

---

## ✅ Improvements I Can Make

- Implement calculation history
- Add custom exception class
- Create Flutter UI version

## 🎯 Conclusion
This project improved my understanding of:
- OOP design

- Error handling

It helped me write safer and more structured Dart code.
